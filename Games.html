Games

Here are some games that I've made, click the pictures for a better look.

Boss Battle

  • I made this on my own using pre made 3D models.
  • The focus for this project was AI.
  • Features a custom made pathfinding algorithm that was a modified version of A*.
  • Heavy use of behaviour trees to accomplish the various agents.
  • I really enjoyed this project and definitly learned a lot. I was very comfortable with finite state machines before so learning about behaviour trees was a lot of fun; combining the two methods gave the agents a great feel in game.

Hotel Hell

  • This was done with a group of three. I did the first two of four scenes as well as helping out on the final scene.
  • The work I did for this project:
    • Created an inventory system.
    • Allowed the player to use items.
    • Produced a system to interact with game world objects.
    • Made a progresion data structure that kept the game logic in tact between scenes.
    • Created a puzzle involving diffent items.
  • This was a fun project, each three of us made diffent puzzles that allowed the player to progress to the next scene.

Drone Racer

  • I made this project on my own, making most of the 3D assets myself.
  • The drone had a fun control scheme that was physics based. The drone was easy to control but very hard to master. After a while, times around the track crould get quite low.
  • This game also had a local split-screen multiplayer function.

Defender Clone

  • This game was 100% made by me in Unity; Code and Art.
  • The game is a clone of popular 80's arcade game Defender.
  • The player has to defend the scientists from a variety of aliens, each with thier own threats.
  • This was my first full game in Unity and it thought me a lot about the engine and how it works.

Rogue Light

  • This game, while simple, was made from scratch in C++ using the SDL library.
  • This was an interesting game to make. I had never used SDL before so there was a lot to learn but I had a great time doing so.
